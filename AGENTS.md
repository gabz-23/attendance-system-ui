# AsistQR — attendance-system-ui

## Stack

- **Framework:** Next.js 16.2.6 (App Router), React 19, TypeScript 5.7 (strict)
- **Styling:** Tailwind CSS v4 (CSS-based config — no `tailwind.config.js`)
- **UI:** shadcn/ui base-nova style + `@base-ui/react` primitives
- **Package manager:** pnpm (not npm)
- **Scaffold:** generated by v0.dev

## Commands

| Command | Notes |
|---|---|
| `pnpm dev` | Start dev server |
| `pnpm build` | Builds silently — `typescript.ignoreBuildErrors: true` in `next.config.mjs` |
| `pnpm lint` | `eslint .` — ESLint not in dependency list; may fail |
| `pnpm add <pkg>` | Use pnpm |

No test runner, test files, or typecheck script exist.

## Architecture

- **Auth:** Supabase Auth with SSR (`@supabase/ssr`). Session managed via httpOnly cookies handled by Supabase middleware. User profile (name, role) stored in `profiles` table synced via DB trigger on signup. Browser client in `lib/supabase/client.ts`, server client in `lib/supabase/server.ts`. Session synced via `onAuthStateChange` in `RoleProvider`.
- **Data:** Supabase database (no mock data). API routes under `/api/auth/*` handle login/register/me/signout using Supabase Auth SDK. All data queries in `lib/queries.ts`.
- **Routing:** App Router — `/` redirects to `/auth/iniciar-sesion`. Dual-role routes under `/professor/*` and `/student/*`.
- **UI language:** Spanish (Mexican dialect).
- **Path alias:** `@/*` maps to project root (`tsconfig.json` `paths`).

## Component conventions

- Components use `data-slot` attributes for shadcn base-nova styling.
- Class merging via `cn()` (`clsx` + `tailwind-merge`) in `lib/utils.ts`.
- Variants via `class-variance-authority` (see `components/ui/button.tsx`).
- Icons from `lucide-react`.
- `components.json` aliases: `@/components`, `@/ui`, `@/lib`, `@/hooks`.

## Notable config quirks

- `next.config.mjs`: `images.unoptimized: true` (no image optimization), `typescript.ignoreBuildErrors: true` (type errors allowed in CI/build).
- `next-env.d.ts` imports `./.next/dev/types/routes.d.ts` — may break if `.next/` is missing.
- `.gitignore` includes v0 trash patterns (`__v0_*`, `.v0-trash/`, `.snowflake/`).
